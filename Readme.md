## ✨ Features
- 🖱️ Move cursor using index finger
- 👌 Pinch → Click
- ✊ Hold pinch → Drag
- ✌️ Two fingers → Scroll
- 🗣️ Sign language → Speech (HELLO, YES, NO)
- ⚡ Real-time (21–35 FPS)

---

## 🧠 Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy
- PyAutoGUI
